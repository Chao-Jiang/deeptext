DeepText
===============================

version number: 0.0.1
author: Yuwen Yan

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install deeptext


Or clone the repo:

    $ git clone https://github.com/ybbaigo/deeptext.git
    $ python setup.py install
